# class-notes

课程文件整理仓库

本仓库用于整理和存储各类课程的笔记和资料，方便学习和复习。

本仓库不仅提供笔记，还提供生成笔记所用的源文件，其中包含 Typst 和 Markdown 格式的文件，方便用户根据需要进行修改和二次编辑。
